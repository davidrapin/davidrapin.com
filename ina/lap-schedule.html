Developper resources

INA's Maven repository.

Direct links to Java Generic LAP Writer library via maven : compiled, sources, javadoc.

The class to extend is fr.ina.dlweb.lap.writer.DefaultLapWriter.

If you use Maven to manage dependencies, add this repository to your pom.xml.

	<repository>
		<id>sonatype-nexus-snapshots</id>
		<url>https://oss.sonatype.org/content/repositories/snapshots</url>
	</repository>

Then, add this dependency to your pom.xml.

	<dependency>
		<groupId>fr.ina.dlweb</groupId>
		<artifactId>lap-writer-generic</artifactId>
		<version>2.0-SNAPSHOT</version>
	</dependency>
